连接mysql

mysql -h{host} -P{port} -u{user} -p’{password}’

示例：mysql -h127.0.0.1 -P3306 -uroot -p’Root@123’

其中：host=1270.0.1或者localhost可省略 port=3306可省略 user=root可省略

即：mysql -p‘Root@123’

查看mysql版本

[root@localhost ~]# mysql -V

或者

mysql> status;

创建数据库

CREATE DATABASE [IF NOT EXISTS] DBNAME [CHARACTER SET ‘CHAR_NAME’][COLLATE ‘COLL_NAME’]

示例：CREATE DATABASE IF NOT EXISTS wxpam CHARACTER SET ‘utf8’ COLLATE ‘utf8_unicode_ci’;

查看数据库的创建语句

SHOW CREATE DATABASE DBNAME;

示例：SHOW CREATE DATABASE wxpam;

创建表

CRTATE TABLE [IF NOT EXISTS] TBNAME(col_name col_definition,…)

示例：

CREATE TABLE t_user (
id INT(10) UNSIGNED NOT NULL AUTO_INCREMENT,
name VARCHAR(32) NOT NULL,
birthday VARCHAR(10) NOT NULL,
cardNo VARCHAR(64) NOT NULL,
createTime DATETIME DEFAULT CURRENT_TIMESTAMP,
updateTime DATETIME DEFAULT NULL ON UPDATE CURRENT_TIMESTAMP,
PRIMARY KEY (id),
INDEX(cardNo),
UNIQUE KEY COMB_UNIQUE (name,birthday),
) ENGINE=INNODB DEFAULT CHARSET=utf8 COLLATE=utf8_unicode_ci;

查看表的创建语句

SHOW CREATE TABLE TABLENAME;

示例：SHOW CREATE TABLE t_user;
